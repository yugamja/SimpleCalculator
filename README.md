# (C# 코딩) 

# 개요
- C# 프로그래밍 학습
- 1줄 소개: C# WinForms 기본 사칙연산 계산기이며 두 텍스트박스에 각각의 표현법으로 연산이 표시된다.
- 사용한 플랫폼:
	- C#, .NET Windows Forms, Visual Studio, GitHub
- 사용한 컨트롤:
	- Label, TextBox, Button
- 사용한 기술과 구현한 기능:
	- 숫자 입력은 Number_Click 이벤트를 통해 처리되며, 버튼의 Text 값을 이용해 입력값을 누적하고, 새로운 연산 이후에는 isNewNumber 변수를 활용하여 기존 입력을 초기화한 뒤 새로운 숫자를 입력받도록 구성함. 이를 통해 사용자 입력 흐름이 자연스럽게 이어지도록 함.
	- 연산자 버튼(+, -, ×, ÷)이 클릭되면 현재 입력값을 int.Parse()를 통해 정수형으로 변환하여 첫 번째 피연산자(num1)에 저장하고, 어떤 연산이 선택되었는지를 문자열 변수(op)에 기록하도록 함. 동시에 expression 변수에 현재까지의 수식을 문자열로 저장하여 txtb_cal에 표시함으로써 사용자에게 연산 진행 상황을 보여줌.
	- (=) 버튼 클릭 시에는 두 번째 입력값을 동일하게 정수형으로 변환한 후, switch문을 이용하여 선택된 연산자에 따라 계산을 수행함. 계산 결과는 ToString() 메서드를 사용하여 문자열로 변환한 뒤 txtb_mscal에 출력하고, txtb_cal에는 “수식=결과” 형태로 완성된 연산 과정을 보여줌.

	- 텍스트박스에 숫자가 입력되도록 함.

## 실행 화면 (과제1)
- 과제2 코드의 실행 스크린샷

![과제2 실행화면](img/task2.png)

- 과제 내용
	- 나머지 사칙연산(빼기, 곱하기, 나누기)을 구현함.

- 구현 내용과 기능 설명
	- (=) 클릭 시 switch문으로 저장된 연산자(op)에 따라 case를 분기하여 덧셈, 뺄셈, 곱셈, 나눗셈 중 해당 연산을 수행하고 결과를 계산하도록 구현함.
	- btn_minus 빼기 기능은 버튼 클릭 시 첫 번째 값num1을 저장하고 op에 '-'를 기록한 뒤, = 버튼에서 두 번째 값num2과의 차를 계산하여 결과를 출력하도록 구현함.
	- btn_times 곱하기 기능은 버튼 클릭 시 입력된 값num1을 첫 번째 피연산자로 저장하고 op에 '*'를 설정한 후, = 버튼에서 두 값을 곱해 결과를 출력하도록 구성함.
	- btn_division 나누기 기능은 버튼 클릭 시 첫 번째 값num1을 저장하고 op에 '/'를 기록한 뒤, (=) 버튼에서 두 번째 값num2으로 나누어 결과를 계산해 출력하도록 구현함.
